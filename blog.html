What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. And the documents represents the structure as a tree. We called the tree as node. Each of node are elements like text, p, div etc allowing scripts to manipulate , access and update the content dynamically.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener on a parent element handles events from its child elements by using event bubbling. It's useful because it improves performance by reducing the number of event listeners and works for dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an elements attributes like setAttribute(),getAttribute. example: element.setAttribute(''); To manipulate style in the DOM like classList. example: element.style.color = 'blue';
